F1 LEAGUE 2023

This game uses race data from F1, which was stored in a pandas data frame using the beautiful soup web scraping package. The data was then stored in a mySQL database, which contains over 400 rows of race data. The data that matters for this game is the drivers points collected at each race. A flask micro framework was coded with python that runs the logic behind the scenes. Html and CSS was used to makes the web pages, while Jinja and javascript are also used. The flask script queries the database live to retrieve the race points chosen by the player and the computer logic.
